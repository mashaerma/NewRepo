# Implementation Plan — Конвертер дат

## 1. Краткое описание задачи
Консольное приложение на C++ для конвертации дат из различных форматов в ISO 8601 и обратно, с ручной реализацией парсера JSON, валидацией данных, генерацией тестовых файлов и замерами производительности.

Проект выполнен в учебных целях для демонстрации работы с файлами, строками, контейнерами STL, обработкой ошибок и бенчмарками.

## 2. Цель и ожидаемый результат
- Работающее CLI-приложение с поддержкой флагов командной строки
- Ручная реализация парсера JSON (объекты, массивы, строки)
- Поддержка 4 форматов дат: YYYY-MM-DD, DD-MM-YYYY, MM/DD/YYYY, DD month YYYY
- Валидация дат (високосные года, корректность дней/месяцев)
- Генератор тестовых JSON-файлов с датами (до 100 000 записей)
- Конвертация в русский длинный формат и обратно
- Замеры производительности и выявление узких мест


## 3. Этапы реализации

| № | Этап | Задачи 

| 1 | Анализ требований | Изучение форматов дат, структуры JSON, требований к лабораторной
| 2 | Проектирование | Разработка структуры проекта, интерфейсов, модулей 
| 3 | Парсер JSON | Ручная реализация чтения JSON, поиска полей, извлечения значений
| 4 | Парсер дат | Реализация функций для 4 форматов, валидация, учёт високосности 
| 5 | CLI интерфейс | Обработка аргументов --help, --input, --output, --to, --generate, --bench
| 6 | Генератор данных | Создание тестовых JSON-файлов с датами (до 100 000 записей) 
| 7 | Тестирование | Написание unit-тестов, проверка граничных случаев 
| 8 | Бенчмаркинг | Замеры времени на разных объёмах данных, выявление узких мест 
| 9 | Документация | README, Implementation Plan, bench.md, test_log.txt | ✅ |
