# Конвертер дат - Лабораторная работа по C++

## Описание
Консольное приложение для конвертации дат из различных форматов в ISO 8601 и обратно.

## Возможности
- Чтение JSON файлов с датами
- Поддержка форматов: YYYY-MM-DD, DD-MM-YYYY, MM/DD/YYYY, "DD месяц YYYY"
- Конвертация в русский и английский длинный формат
- Валидация и обработка ошибок
- Сохранение результатов в JSON
